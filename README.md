# Bore-TestRepo2

Static website for BORE Inc. — a company that does things.

## Pages

- `index.html` — Homepage
- `about.html` — Company info and team bios
- `services.html` — What we sell
- `contact.html` — How to reach us
- `404.html` — For when things go wrong

## Structure

Plain HTML + one CSS file (`style.css`). No build step, no dependencies, no excuses.

## Branches

- `main` — production
- `staging` — pre-release integration
- `feature/*` — new features
- `bugfix/*` — bug fixes
- `hotfix/*` — urgent fixes
- `chore/*` — maintenance tasks
